# Topcoder Utils

A maven demo.

### 概 述

基于Maven实现Utils工具类。主要功能有：

* 日志打印工具类
* `.properties`文件相关操作工具类
* 变量格式验证工具类

访问：[**JavaDoc**](./docs/javadoc/index.html)
